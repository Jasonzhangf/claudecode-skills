# References模板和标准 - 提供项目文档标准化模板和分析标准

## 结构化功能描述（第一行要求）
**模块功能的一句话结构化描述** - 必须在第一行用一句话清晰描述该模块的核心功能，格式为：功能描述 - 主要职责 - 适用场景

## 模块概述
references模块为sysmem项目提供标准化的文档模板、架构分析标准和规则定义，确保项目文档的一致性和分析结果的准确性。

## 文件结构
列出模块中的每个文件及其功能说明：

### 核心模板文件
- `readme_template.md` - 标准README模板，确保所有模块文档格式一致性
- `claude_md_template.md` - CLAUDE.md结构模板和示例，定义项目架构文档标准

### 分析标准文件
- `analysis_criteria.md` - 架构分析标准和规则定义，包含文件重复检测、代码相似度计算、文档质量评估等标准

## 重要定义（Ground Truth）
### 核心功能定义
**重要**: references模块只提供模板和标准，不包含具体实现逻辑，确保标准化和一致性

### 接口定义
**重要**: 所有模板必须遵循统一的格式规范，包括第一行功能描述、文件结构说明、重要定义标记等

### 数据结构定义
**重要**: 分析标准提供量化的评估指标和明确的检测规则，支持自动化的架构分析
