# 情节模板

## 故事结构模板

### 三幕式结构模板
```markdown
# [故事标题] 情节大纲

## 第一幕：开端 (第1-10章)
**目标**: 建立世界观，介绍角色，设置基本冲突

### 章节安排
- **第1-2章**: 主角登场，展示日常生活状态
- **第3-4章**: 激励事件出现，打破平衡
- **第5-6章**: 主角面对选择，决定行动
- **第7-8章**: 开始行动，遇到初步困难
- **第9-10章**: 第一幕结束，进入新世界

### 重要情节点
1. **日常展示**: [展示主角的日常生活]
2. **激励事件**: [触发故事的关键事件]
3. **决定时刻**: [主角决定行动的时刻]
4. **跨过门槛**: [正式进入故事主线]

## 第二幕：发展 (第11-30章)
**目标**: 冲突升级，角色成长，情节推进

### 上半部分 (第11-20章)
- **第11-15章**: 适应新环境，学习新技能
- **第16-20章**: 遇到盟友，面对初步挑战

### 中点转折 (第21-22章)
- **重大发现**: [改变故事走向的重要发现]
- **策略调整**: [根据新情况调整策略]

### 下半部分 (第23-30章)
- **第23-26章**: 困难加剧，遭遇挫折
- **第27-30章**: 准备最终对决

### 重要情节点
1. **初步成功**: [取得的小胜利]
2. **重大挫折**: [遭遇的重大失败]
3. **中点转折**: [故事中期的重大转折]
4. **最后准备**: [为最终对决做准备]

## 第三幕：高潮与结局 (第31-40章)
**目标**: 达到高潮，解决冲突，收束情节

### 高潮部分 (第31-35章)
- **第31-33章**: 最终对决开始
- **第34-35章**: 战斗高潮，解决主要冲突

### 结局部分 (第36-40章)
- **第36-37章**: 处理后续影响
- **第38-39章**: 角色归宿，情感收束
- **第40章**: 结局，主题升华

### 重要情节点
1. **最终对决**: [解决核心冲突的最终战斗]
2. **问题解决**: [主要问题的解决方式]
3. **角色成长**: [主角最终的成长状态]
4. **主题展现**: [故事主题的最终表达]
```

### 英雄之旅模板
```markdown
# [故事标题] 英雄之旅大纲

## 第一阶段：启程
1. **平凡世界**: [主角的日常生活]
2. **冒险召唤**: [故事的触发事件]
3. **拒绝召唤**: [主角的犹豫和拒绝]
4. **与智者相遇**: [获得指导和帮助]
5. **跨越第一个阈限**: [决定踏上旅程]

## 第二阶段：启蒙
6. **考验、盟友和敌人**: [遇到的各种考验]
7. **深入洞穴**: [接近最终挑战]
8. **严峻的考验**: [最大的考验和磨难]
9. **获得奖赏**: [从考验中获得的东西]

## 第三阶段：归来
10. **回归之路**: [开始回家的旅程]
11. **复活**: [最终的重生和成长]
12. **携万能药回归**: [带回的宝贵收获]
```

## 冲突设计模板

### 内心冲突模板
```markdown
# [角色名] 内心冲突

## 冲突双方
- **欲望A**: [内心的第一种欲望]
- **欲望B**: [内心的第二种欲望]
- **冲突原因**: [两种欲望冲突的原因]

## 冲突表现
- **行为矛盾**: [行为上的矛盾表现]
- **情绪波动**: [情绪的起伏变化]
- **决策困难**: [做决定时的困难]

## 冲突发展
- **加剧因素**: [加剧冲突的因素]
- **转折点**: [冲突发展的转折点]
- **最终选择**: [最终的艰难选择]

## 冲突解决
- **解决方式**: [冲突的解决方式]
- **成长收获**: [从冲突中获得的成长]
- **新的平衡**: [达到的新平衡状态]
```
