# 角色模板

## 基本信息模板

### 主角模板
```markdown
# [角色姓名]

## 基本信息
- **年龄**: [具体年龄或年龄段]
- **性别**: [男/女/其他]
- **职业**: [当前职业或身份]
- **身份**: [在社会中的地位]

## 外貌特征
- **身高**: [具体身高]
- **体型**: [体型描述]
- **面容**: [面部特征]
- **发型**: [发型和发色]
- **着装风格**: [日常穿着风格]
- **特殊标记**: [伤疤、纹身等特殊标记]

## 性格特点
- **核心性格**: [最主要的性格特征]
- **优点**: [列举3-5个主要优点]
- **缺点**: [列举3-5个主要缺点]
- **习惯**: [日常习惯和动作]
- **口头禅**: [常说的话]

## 背景故事
- **家庭背景**: [家庭成员和成长环境]
- **教育经历**: [学习和成长经历]
- **重要经历**: [影响人生的重要事件]
- **创伤经历**: [可能影响行为的创伤]

## 能力特长
- **专业技能**: [掌握的专业技能]
- **天赋才能**: [天生的才能]
- **特殊能力**: [如果有的特殊能力]
- **学习经验**: [通过学习获得的经验]

## 目标动机
- **短期目标**: [近期想要实现的目标]
- **长期目标**: [人生的终极追求]
- **内在动机**: [驱动行为的内在原因]
- **外在压力**: [来自外部的压力和期待]

## 人际关系
- **家人**: [与家人的关系]
- **朋友**: [重要的朋友关系]
- **敌人**: [敌对关系]
- **恋人**: [恋爱关系]
- **导师**: [指导和帮助的角色]

## 角色发展弧线
- **起点**: [故事开始时的状态]
- **转变点**: [重要的转变时刻]
- **终点**: [故事结束时的成长]

## 经典台词
- [代表性台词1]
- [代表性台词2]
- [代表性台词3]
```

### 配角模板
```markdown
# [角色姓名]

## 基本信息
- **角色定位**: [在故事中的作用]
- **与主角关系**: [与主角的具体关系]
- **出场频率**: [经常/偶尔/一次性]

## 性格特征
- **主要特点**: [1-2个突出特点]
- **对主角的影响**: [如何影响主角]

## 作用功能
- **推动情节**: [如何推动故事发展]
- **衬托主角**: [如何衬托主角]
- **提供信息**: [提供什么信息]
- **制造冲突**: [制造什么冲突]
```

## 角色关系模板

### 关系图谱模板
```markdown
# 角色关系图

## [主角A]
- **[角色B]**: [关系描述] - [关系强度: 强/中/弱] - [情感倾向: 正面/负面/复杂]
- **[角色C]**: [关系描述] - [关系强度: 强/中/弱] - [情感倾向: 正面/负面/复杂]

## [角色B]
- **[角色C]**: [关系描述] - [关系强度: 强/中/弱] - [情感倾向: 正面/负面/复杂]
```

### 关系发展模板
```markdown
# [角色A] 与 [角色B] 关系发展

## 初始状态
- **关系类型**: [陌生人/认识/朋友等]
- **情感基础**: [初始的情感态度]
- **互动模式**: [如何相互对待]

## 发展过程
- **转折点1**: [关系变化的关键事件]
- **转变原因**: [导致变化的原因]
- **新状态**: [关系进入新阶段]

## 最终状态
- **关系类型**: [最终的关系类型]
- **情感深度**: [情感的深厚程度]
- **未来展望**: [关系的发展趋势]
```

---
*使用说明: 根据角色类型选择相应模板，填写详细信息。可以删除不适用的部分，补充需要的内容。*
